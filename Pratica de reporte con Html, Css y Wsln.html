Primer Comando

Vim

Es un editor de texto minimalista que se puede usar en Linux. Vim destaca por su potencia y versatilidad, proporcionando una interfaz eficiente y robusta para la edición de texto en entornos de línea de comandos.


Segundo Comando

Nano

Nano es un editor de texto para la terminal en sistemas Unix y Linux. Destacado por su simplicidad y facilidad de uso, es ideal para usuarios principiantes. Permite crear, editar y visualizar archivos de texto directamente desde la línea de comandos.

  • Para entrar desde linux: ~$ nano nombre_del_archivo.txt
  • Para entrar al archivo: Ctrl + O
  • Para guardar el archivo: Ctrl + X
  • Para salir de Nano: Ctrl + X
  • Para deshacer la última acción: Ctrl + U

Tercer Comando

Micro

Micro es un editor de texto para la terminal en sistemas Unix. Destacado por su simplicidad y potencia, ofrece una interfaz fácil de usar con características esenciales para la edición de archivos de texto en línea de comandos.

  • Para invocar Micro en la terminal: ~$ micro nombre_del_archivo.txt
  • Para entrar al archivo: Ctrl + E
  • Para guardar el archivo: Ctrl + S
  • Para salir de Micro: Ctrl + Q
  • Para buscar texto: Ctrl + F

Cuarto Comando

Cat

Cat es un comando en Unix y sistemas Unix-like que muestra el contenido de archivos de texto en la terminal. Es útil para ver archivos pequeños, concatenar y crear archivos nuevos.

  • Para ver el contenido de un archivo: ~$ cat nombre_del_archivo.txt
  • Para concatenar varios archivos y mostrar su contenido: ~$ cat archivo1.txt archivo2.txt
  • Para crear un nuevo archivo: ~$ cat > nuevo_archivo.txt
  • Para añadir contenido a un archivo existente: ~$ cat >> archivo_existente.txt
