Comando para Listar carpetas y documentos

  • Como se escribe el comando: ls o tree

    • La funcion de estos es mostar de una forma ordenada las carpetas y documentos que hemos hecho, ls los muestra de una forma sencilla minetras que tree lo hace de una forma mas bonita

  • Que hace la opcion: ls -l

    • Nos permite ver los permisos que tiene el archivo

  • Que hace la opcion: ls -a

    Nos permite listar todos los archivos y directorios, incluyendo los ocultos, y poder verlos


Comando para crear un documento

El comando para crear un documento es: ~$ touch Nombre_del_archivo.txt



Comando para crear un directorio

El comando para crear un directorio es: ~$ mkdir Nombre_del_directorio


Comandos para eliminar

  • El comando para eliminar un documento es: ~$ rm Nombre_del_archivo
  • El comando para eliminar un directorio es: ~$ rm -r Nombre_del_directorio

Comando para modificar un archivo en nano

  • Como se guardan cambios: Ctrl + O
  • Como se cierra el editor: Ctrl + X
